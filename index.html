Projeto 1

Este projeto foi desenvolvido para apresentar a uma turma o que seria desenvolvido durante o curso. O site contém os principais cantores, com seus respectivos albuns de música, contendo também link para spotify e telas dedicadas para cada cantor, contando a trajetória de cada um.

Projeto 2

Desenvolvi um site responsivo para uma pizzaria, focado em oferecer uma experiência de usuário agradável e facilitar pedidos online. POssui um design responsivo com um menu interativo, exibindo o nome e preço do produto atráves de um hover.

Projeto 3

Desenvolvi uma funcionalidade simples para alternar entre temas claro e escuro em um site. O objetivo do projeto foi melhorar a acessibilidade e a personalização da experiência do usuário, permitindo que eles escolham o tema que preferem.

Projeto 4

Foi desenvolvido um sistema de administração completo para gerenciamento de conteúdo e usuários. O objetivo do projeto foi criar uma plataforma intuitiva e eficiente que permita a administração centralizada de diversas funcionalidades do site.
